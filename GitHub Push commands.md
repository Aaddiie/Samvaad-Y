# GitHub Push commands



### **1. Fresh Repo (first time push)**

Step 1: Initialize git

git init

Step 2: Add all files

git add .

Step 3: Commit

git commit -m "initial commit"

Step 4: Add remote repo (copy from GitHub)

git remote add origin https://github.com/your-username/your-repo.git

Step 5: Set branch (important)

git branch -M main

Step 6: Push

git push -u origin main





### **2. Update Existing Repo (after changes)**

Step 1: Check changes

git status

Step 2: Add changes

git add .

Step 3: Commit

git commit -m "your message"

Step 4: Push

git push
